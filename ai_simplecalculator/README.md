# AI Hybrid Calculator - Dark Futuriste

Une application de calculatrice **hybride**, avec interface **Dark Futuriste** et mode **AI**.  
Vous pouvez effectuer des calculs classiques avec les boutons ou écrire une phrase en langage naturel pour obtenir le résultat.

---

## Fonctionnalités

- Mode **AI** : écrire `calcule 12 + 5` → résultat affiché
- Mode **Classique** : boutons chiffres et opérations (+, -, \*, /, =, C, ⌫, +/-)
- Interface Dark Futuriste avec design inspiré IA / Hacker
- Fenêtre fixe, simple et élégante
- Mode AI simple : comprend des phrases naturelles pour calculer des opérations arithmétiques.

---

## Installation

1. Cloner ou télécharger le projet
2. Installer **Python 3.8+**
3. Installer **PyInstaller** (pour créer l’exécutable) :

```bash
pip install pyinstaller
```
